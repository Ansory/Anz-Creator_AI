Tempatkan file MP3 BGM di folder ini dengan nama sesuai mood:
- cinematic.mp3  (epic, dramatis)
- happy.mp3      (ceria, upbeat)
- dark.mp3       (horror, mystery)
- motivational.mp3 (inspiring)
- romantic.mp3   (soft, emotional)
- action.mp3     (energetic, fast)

Rekomendasi sumber royalty-free:
- https://pixabay.com/music/
- https://www.bensound.com/
- YouTube Audio Library (dengan atribusi)

Jika file tidak ada, video tetap di-render tanpa BGM.
